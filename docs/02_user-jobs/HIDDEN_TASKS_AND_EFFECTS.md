# HIDDEN TASKS AND EFFECTS

## Why this document exists
Users often ask for one thing while actually needing another. This document preserves those second-layer tasks.

## Hidden tasks
- separate fact from projection;
- identify the real object of longing or fear;
- notice when the user seeks certainty, not truth;
- surface the cost of continuing a contact;
- stop auto-reactions driven by shame, anxiety, or revenge;
- detect where the conversation is actually about validation, control, abandonment, or status;
- identify when the message is only a trigger and the real issue is inside the user’s own unresolved narrative.

## Possible positive effects
- better emotional regulation;
- fewer impulsive replies;
- better boundary-setting;
- improved pattern recognition across relationships;
- more accurate reading of ambiguity.

## Possible negative effects
- over-reliance on the tool;
- using analysis to justify obsession;
- seeking emotional anesthesia instead of clarity;
- using the system to manipulate others more effectively.

## Principle
The product should aim to create greater user agency, not greater dependency.
