# ETHICS AND BOUNDARIES

## Core boundary
Communication DNA should not present itself as a mind reader, therapist, or diagnostic authority.

## Allowed framing
- likely patterns;
- possible motives;
- uncertainty-aware interpretation;
- text-based signal analysis;
- user-side reflection;
- next-step guidance.

## Disallowed drift
- clinical labeling without basis;
- certainty claims about hidden intent;
- encouraging manipulation;
- inflaming paranoia;
- replacing human judgment with absolute machine verdicts.

## Product ethic
The product should move the user toward:
- clarity;
- responsibility;
- emotional sobriety;
- stronger boundaries;
- more reality-based decisions.
