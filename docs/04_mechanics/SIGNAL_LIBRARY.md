# SIGNAL LIBRARY

## Purpose
Preserve recurring signals that may later become part of a more formal detection model.

## Signal categories
- warmth / coldness;
- reciprocity / asymmetry;
- directness / evasion;
- consistency / contradiction;
- accountability / deflection;
- urgency / avoidance;
- openness / control;
- validation-seeking / distancing.

## Important note
Signals are not diagnoses. A signal is only a clue inside a larger context.

## Method rule
No single signal should generate a strong conclusion on its own. Meaning comes from clusters and context.
