# ANALYSIS MODEL

## Working output model
A strong analysis should separate layers instead of blending everything into one answer.

### Layer 1. Observable signals
What is actually present in the text:
- wording;
- tone;
- intensity;
- avoidance markers;
- contradictions;
- reciprocity level;
- time/effort asymmetry;
- pressure or distancing cues.

### Layer 2. Probable hidden dynamics
Possible motives, needs, fears, or defensive strategies inferred from the signals.
These should always be framed as hypotheses, not facts.

### Layer 3. User-side reading risk
Where the user may be projecting, catastrophizing, idealizing, or seeking certainty.

### Layer 4. Position
What is the user’s actual position in this dynamic right now.

### Layer 5. Next move
A small set of distinct next moves:
- clarify;
- pause;
- test reciprocity;
- set a boundary;
- de-escalate;
- exit;
- do not answer yet.

## Meta-principles
- separate signal from story;
- preserve uncertainty;
- show confidence level;
- avoid pathologizing;
- help the user return to position and agency.
