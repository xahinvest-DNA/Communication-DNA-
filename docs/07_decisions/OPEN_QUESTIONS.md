# OPEN QUESTIONS

## Product questions
- What should be the first wedge: incoming message analysis, draft checking, or internal monologue analysis?
- Should the system recommend one move or several?
- How much depth is useful in a high-emotion state?
- How should confidence be shown?

## Positioning questions
- Is the product framed as clarity, communication intelligence, emotional navigation, or hidden-structure analysis?
- How explicit should the language of hidden need and motive be?

## Format questions
- Bot first, app first, or hybrid?
- Single-session tool or profile-building system?

## Safety questions
- How to reduce dependency?
- How to prevent manipulation use cases?
- How to handle privacy-sensitive user trust?
