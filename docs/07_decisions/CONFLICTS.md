# CONFLICTS

## Why this file exists
Not all important tensions should be prematurely resolved. Some conflicts are structurally useful and need to remain visible.

## Current conflicts
- precision vs accessibility;
- depth vs emotional overload;
- clarity tool vs addictive interpretation engine;
- user agency vs external authority dependence;
- broad personal territory vs narrow MVP wedge;
- emotionally resonant language vs overclaiming.

## Rule
A conflict should stay here until there is enough evidence or reasoning to resolve it, not until it becomes inconvenient.
